# FNLLA Web brand assets

This directory stores the committed FNLLA Web logo source and the generated preview images used for repository presentation.

Included files:

- `fnlla-ui.svg`: official logo source provided for FNLLA Web
- `fnlla-ui-dark.svg`: dark-mode logo variant for dark surfaces and repository assets
- `fnlla-github.svg`: simplified organization-avatar mark without the `WEB` badge
- `fnlla-github.png`: GitHub-ready organization avatar export

The `WEB` badge is no longer embedded directly inside the core `fnlla-ui` logo files. Where needed, it should be generated separately by code so the underlying logo asset remains clean and reusable.
